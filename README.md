# DrawFight Web

`Draw Survivors` 是一个以攻击池抽卡为核心的幸存者类网页游戏原型。

## 运行

直接打开根目录的 `index.html`，选择版本即可开始。项目为纯静态文件，也可以直接部署至 GitHub Pages。

## 操作

- `WASD` / 方向键：移动
- `Esc`：暂停或继续
- `1` / `2` / `3`：选择升级

## 项目结构

- `index.html`：版本选择
- `versions/v1/index.html`：V1 游戏入口
- `versions/v1/game.js`：V1 游戏逻辑
- `versions/v1/styles.css`：V1 样式
